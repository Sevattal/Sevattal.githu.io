Nirsoft 工具集合介绍

官网地址：https://www.nirsoft.net/

前言

NirSoft 提供了将近 200 个绿色免费而且实用的小工具，包含“密码恢复工具”、“网络监控工具”、“web浏览器工具”、“音视频相关工具”、“互联网相关工具”、“桌面实用工具”、“outlook/Office工具”、“编程工具”、“磁盘工具”、“系统工具”以及“其他实用工具”。这些工具多数都支持中文语言，不少独立博客和网站都不同程度的介绍过它们。

作者 NirSofer ，一个超级勤奋（长达十多年不间断地制作和更新程序）的程序员，他的官网上介绍自己说，具有 C++ 和 .net 开发经验，熟悉 Windows API 和加解密算法。NirSoft 是一个人的网站，没有开发队伍。只是作者一个人利用业余时间开发的工具，还要坚持维护网站、修复程序 bug ，回答一部分来自世界各地的电子邮件。尽管网站需要空间和域名费用，但这将近 200 个小工具都是免费的，不含广告、大多数程序的配置文件保存为同名 .cfg 文件，不写入注册表。最牛逼的是多数程序都支持一个独特的翻译系统，只要采用 /savelangfile 参数运行程序，就可以生成 后缀为 lng.ini 的文件，任何人都可以翻译这个文本文件，以实现程序的本地化，比如中文。
