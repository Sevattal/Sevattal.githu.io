SQLite-Introduction-1

SQLite 入门及简单使用

参考文档：https://www.cnblogs.com/teroy/p/3960760.html

什么是SQLite

SQLite 是一款非常轻量级的关系数据库系统，支持多数 SQL92 标准。SQLite 在使用前不需要安装设置，不需要进程来启动、停止或配置，而其他大多数 SQL 数据库引擎是作为一个单独的服务器进程，被程序使用某种内部进程通信(典型的是 TCP／IP )，完成发送请求到服务器和接收查询结果的工作，SQLite 不采用这种工作方式。使用 SQLite 时，访问数据库的程序直接从磁盘上的数据库文件读写，没有中间的服务器进程。使用 SQLite 一般只需要带上一个 dll ，就可以使用它的全部功能。
SQLite 的主要应用场景有作为手机应用的数据库以及小型桌面软件的数据库。
