Linux-Kali-Install-1

Linux

Kali 作为主力办公机后必做的一些事

前言

kali 本身是一个 Debain 系统对于网络安全工作的一个改版系统,内置很多网络安全工具。

GNOME 界面（本人较为喜欢的一个linux桌面）

1.下载安装gnome

apt -y install kali-desktop-gnome

注意：要是安装不了请更新 apt update

2.安装过程中会弹出界面，选择gdm3，按回车键

3.安装完成后，只需要重启选择 GNOME 桌面即可

安装中文输入法

1.安装输入法输入法框架

apt-get install fcitx

2.下载好以后，程序中会多出下面两个图标

Fcitx 、 Fcitx Configuration

3.现在安装一个中文输入法 google 输入法

apt-get install fcitx-googlepinyin

3.运行以下命令选择系统输入法为 fcitx 输入法

im-config

4.点击 fcitx 配置，可以看到 Google 拼音，如果没有点击左下角的 + 进行搜索

注：若找不到 Google Pinyin 则需要重启操作系统。

切换界面语言

1.打开Terminal Emulator 界面，查看当前系统语言为默认英文
2.在Terminal Emulator 中执行 dpkg-reconfigure locales命令
3.使用空格键取消勾选en_US.UTF-8 UTF-8选项
4.勾选 [ ]zh_CN.GBK_GBK 和 [ ] zh-CN.UTF-8.UTF-8 两个选项
5.使用reboot命令重启机器使其配置生效

网络防火墙设置

1.安装ufw

apt-get install ufw

2.关闭防火墙

ufw disable

3.开启防火墙

ufw enable

防火墙常用命令如下

检查防火墙的状态（默认 inactive） # ufw status
防火墙版本 # ufw version
启动ufw防火墙 # ufw enable
关闭ufw防火墙 # ufw disable
默认禁止访问所有 # ufw default deny
开放22/TCP端口 # ufw allow 22/tcp
开放53端口(tcp/udp) # ufw allow 53
禁止外部访问 # ufw deny 3306
删除已经添加过的规则 # ufw delete allow 22
允许此IP访问所有的本机端口 # ufw allow from 192.168.1.100
删除上面的规则 # ufw delete allow from 192.168.1.100
查看规则，显示行号 # ufw status numbered
删除第三条规则 # ufw delete 3
关闭ufw # ufw disable
禁止对8888端口的访问 # ufw deny 8888
打开来自192.168.0.1的tcp请求的22端口 # ufw allow proto tcp from 192.168.0.1 to any port 22

安装 WPS (目前来说 WPS 国际版各方面比较优秀)

WPS 目前国际版关闭后没有后台进程，运行时对比其他几款 Word 处理软件来说使用内存更少，并且提供了 deb 和 rpm 等安装包。

1.下载

下载地址：https://www.wps.com/（去WPS官网下载）

2.安装

sudo dpkg -i wps-office_10.1.0.5672~a21_amd64.deb

安装 Chrome 浏览器

从浏览器角度 Kali 自带了 Firefox,对于信仰 Chrome 的人来说还是安装 Chrome 吧。
Chrome 的安装也是比较方便的,支持 deb 和 rpm 等安装包。

1.下载

下载地址：https://www.google.cn/chrome/（去Chrome官网下载）

2.安装

sudo dpkg -i google-chrome-stable_current_amd64.deb

3.命令行启动

google-chrome

注：当然 google 安装好后有对应的桌面图标，此处只是列出命令行启动 Chrome 命令

其他开发软件列出

前端开发软件： vscode (deb、rpm安装包)
数据库软件：dbeaver-ce (deb、rpm安装包)
后端开发软件：IntelliJ IDEA (linux 免安装包)
